- Configuring Git
git config --global user.name "username"
git config --global user.email "email"
git config user.name
git config user.email

- Initialize a Repository
git init

- Check the status of all of our files and folders
git status

- GIT 3 STAGE
1. workspace - untracked files
2. staging - add files
- Add all of the files that have been modified to the staging area
git add .
2. commit 
- Create a commit


- Check the history
git log 
git log --oneline


- BRANCHING

- List all of the branches
git branch -a

- Create a branch
git branch [branch-name]

- Switch to a different branch
git switch [branch-name]
git checkout [branch-name] - for older version of git
